# 3D Drawing Application

## Introduction
This 3D Drawing Application allows users to create and manipulate three-dimensional objects using an intuitive interface.

## Features
- **3D Object Creation**: Users can create various geometric shapes like cubes, spheres, and custom designs.
- **Manipulation Tools**: Tools for resizing, rotating, and positioning objects within the 3D space.
- **Camera Controls**: Navigate through the 3D environment with pan, zoom, and rotation capabilities.
- **Export Options**: Save your models in various formats for 3D printing or further editing.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Sk2000f/3d-drawing-app.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```

## Usage
1. Start the application and create a new project.
2. Use the toolbar to select shapes and tools.
3. Click and drag to draw shapes or manipulate existing objects.
4. Save your work frequently!
